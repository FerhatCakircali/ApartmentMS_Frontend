1 - Backend kısmında gerekli işlemleri yaptıktan sonra terminal'e

npm install
npm start 
 
yazın.

2 - Açılan " http://localhost:3000/ " login ekranına yönlendirecektir.


3 - Burada default olarak backend kısmında oluşturmuş olduğumuz admin hesabı vardır.


4 - kullanıcı adı = admin ve şifre = 21 

olarak login yapıyoruz.
